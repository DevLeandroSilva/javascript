Operadores lógicos
! Negação
&& Conjunção - Operador binário 
Se os dois forem verdadeiros, tudo será
true, caso contrário, tudo será false.
Ex:
true && true =  true
true && false =  false
false && true = false
false && false = false
 
|| Disjunção - Operador binário com 
doi valores lógicos
Ex.:
true || true = true
true || false = true
false || true = true
false || false = false
